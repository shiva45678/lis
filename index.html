Integration Flow

How It Connects

We support both simple output capturing and full closed-loop automation.

1 Sample Processing

Technician manually programs the analyzer or scans the barcode. The machine runs the required assay/test.

2 Data Capture

Once complete, the analyzer transmits raw string data (via RS232 or TCP/IP). Our middleware listens and captures the packet.

3 HMS Push

The software decodes the ASTM/HL7 string into structured parameters and pushes it to the patient's digital record in the HMS.

1 Order Request

Doctor prescribes a test in the HMS. The middleware automatically pushes a Work Order directly to the machine.

2 Barcode Scan

Technician scans the sample barcode. The analyzer already knows which tests to run without manual data entry.

3 Auto Validation

Results are sent back, checked for critical flags by the middleware, and posted instantly to the doctor's HMS dashboard.
